# 🛡️ Web-Based Honeypot System

A web-based honeypot that simulates realistic authentication portals and captures user interactions for security analysis. The system integrates Large Language Model (LLM) APIs to generate dynamic responses, creating realistic attacker interactions while providing monitoring dashboards for analysis.

---

## Features

- Fake login and authentication pages
- Dynamic LLM-generated responses
- Dashboard for monitoring captured interactions
- User activity logging
- Git version control
- Web deployment

## Installation

Clone the repository

```bash
git clone https://github.com/yourusername/web-honeypot.git
```

Move into the project directory

```bash
cd web-honeypot
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```
